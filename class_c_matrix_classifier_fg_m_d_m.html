Classify the matrix and return the class id, the distance and the probability of each class.

  1. Transform the sample to the Tangent Space.
  2. Apply the FgDA weight.
  3. Return to the original Manifold.
  4. Apply the MDM classify (CMatrixClassifierMDM::classify(const Eigen::MatrixXd&, size_t&, std::vector<double>&, std::vector<double>&, const EAdaptations, const size_t&)).

Remark : We use the MDM classification whatever the adaptation method chosen. Thus the MDM part evolves but the geodesic filtering does not evolve to keep an execution online. A version allowing the adaptation of the Filter will be implemented for offline execution.

Parameters
sampleThe sample to classify.
classIdThe predicted class.
distanceThe distance of the sample with each class.
probabilityThe probability of the sample with each class.
adaptationAdaptation method for the classfier EAdaptations.
realClassIdThe expected class id if supervised adaptation.
Returns
True if it succeeds, false if it fails.

bool CMatrixClassifierFgMDM::train ( const std::vector< std::vector< Eigen::MatrixXd >> &  datasets)
overridevirtual

Train the classifier with the dataset.

  1. Compute the Riemann mean of all trials as reference and store this in m_Ref member.
  2. Set the good number of classes
  3. Trasnform data to the Tangent Space with the reference
  4. Compute the FgDA Weight (FgDACompute).
  5. Apply the FgDA Weight and return to Original Manifold.
  6. Apply the MDM train ( CMatrixClassifierMDM::train(const std::vector<std::vector<Eigen::MatrixXd>>&)).
Parameters
datasetsThe data set one class by row and trials on colums.
Returns
True if it succeeds, false if it fails.
